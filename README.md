# taka-workspace

ドキュメント・ワークフロー・AI 協奏のためのワークスペース

## 📁 フォルダ構成

```
taka-workspace/
├── docs/           # ドキュメント・設計書
├── workflows/      # ワークフロー・手順
├── memory/         # メモ・記録
├── tools/          # スクリプト・ツール
└── README.md
```

## 🚀 使い方

- **docs/**: 設計、ガイド、決定事項などを管理
- **workflows/**: GitHub、Vercel、AI 連携などの手順
- **memory/**: セッションごとのメモ、学びの記録
- **tools/**: 自動化スクリプト、ユーティリティ
