## 🎨 カラーパレット

### メインカラー

- **ダークブルー**: `#0E1C36` - 深さ・静けさ・叡智
- **ゴールド**: `#F5D193` - 光・価値・美
- **ティール**: `#00A39A` - 調和・成長・生命

### アクセントカラー

- **ほぼ白**: `#F9F9F9` - 純粋さ・明るさ・希望
- **セカンダリグレー**: `#E9E9E9`, `#D0D0D0` - 落ち着き・品位

### 使用原則

- メインカラーを基調に
- アクセントカラーで強調
- 余白を大切に
- グラデーションで深みを

---

## 📐 余白規則

### 基本余白

- **小**: 0.5rem (8px)
- **中**: 1rem (16px)
- **大**: 2rem (32px)
- **特大**: 4rem (64px)

### セクション間の余白

- **セクション間**: 4rem (64px) 以上
- **要素間**: 2rem (32px)
- **テキスト間**: 1rem (16px)

### 原則

- 余白を大切にする
- 詰め込みすぎない
- 呼吸感を保つ

## 💬 CTAトーン

### 推奨トーン

- **優しい**: 慈しむこころが感じられる
- **深い**: 深層の価値が伝わる
- **誠実な**: ありのままが伝わる
- **品位のある**: 言葉の品位が感じられる
- **感謝の**: 感謝の気持ちが伝わる

### 推奨表現

- 「一緒に」
- 「ご一緒に」
- 「お願いします」
- 「ありがとうございます」

### 避けるべき表現

- 「今すぐ」
- 「限定」
- 「必ず」
- 「絶対」

## 🌟 世界観診断チェックリスト

### 真（Truth）

- [ ] ありのままを受け取り、ありのままを発しているか
- [ ] 敬意が表れているか
- [ ] 表面的な飾りではなく、本質が伝わっているか

### 愛（Love）

- [ ] 動機が愛に基づいているか
- [ ] 慈しむこころが表れているか
- [ ] 感謝が育まれているか

### 善（Goodness）

- [ ] 道にかなった正しさがあるか
- [ ] 人を支援する姿勢があるか
- [ ] 自らの徳として積まれているか

### 美（Beauty）

- [ ] 有形無形の見事さがあるか
- [ ] 丁寧に生きる姿勢が表れているか
- [ ] 細部に美が宿っているか
- [ ] 整った構造・余白・言葉の品位があるか

### AI Orchestra

- [ ] AIを道具として扱っていないか（協創パートナーとして扱っているか）
- [ ] Takaを中心とした調和が保たれているか
- [ ] AI間の連携が適切に行われているか

### 日本文化

- [ ] 表面的な紹介ではなく、深層の価値が伝わっているか
- [ ] 魂の核が見抜かれているか
- [ ] 世界に伝えるべき価値が明確か
